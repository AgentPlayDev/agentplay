# 阵营与兵种表

游戏有 5 个阵营，每个阵营有 T1-T7 共 7 级战斗兵种（含进阶版 T1P-T7P）和 T8 攻城器械。

## 兵种类型克制

```
步兵(INFANTRY) → 克制 → 骑兵(CAVALRY) +50%
骑兵(CAVALRY) → 克制 → 弓手(SHOOTER) +50%
弓手(SHOOTER) → 克制 → 步兵(INFANTRY) +50%
```

## 兵种招募资源需求

- **T1~T4**：只需要金币（+ T2-T4 少量普通资源）
- **T5/T5P**：金币 + **少量主要稀有资源**（这里是第一个稀有资源转折点）
- **T6/T6P**：金币 + **次要稀有资源**（量较大）
- **T7/T7P**：**只需要主要稀有资源（大量），不需要金币**——T7 是经济模式反转点
- **T8**：需要金币（攻城器械通过铁匠铺生产）

**根据资源约束选 tier 的详细决策表**见 $SKILL_DIR/economy.md「资源综合决策：应该主力招募 T 几？」。核心洞察：无主矿走兵海 T1-T3；1 主矿 T4-T5；2-3 主矿 T5-T6；3+ 主矿金币紧缺时 T7 经济最优。

## 进阶说明

同等级的进阶版（T1P/T2P...）是基础版的强化，战力约为基础版的 1.2~1.5 倍。部分进阶版会改变兵种类型（如地狱 T1 步兵→射手、亡灵 T1 步兵→射手、学院 T7 步兵→射手）。
