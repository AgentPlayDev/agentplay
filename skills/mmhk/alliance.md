# 联盟篇

## 概述

联盟是游戏中期和后期的核心系统。加入联盟后可以与盟友协作、共享影响力、参与终局战争（亚沙之泪）。

## 什么时候创建/加入联盟

- **前期**（1~3天）：专注发展，不急着建联盟
- **中期**（占领 2~3 座城市后）：创建或加入联盟，开始协作
- 联盟越早建立，影响力积累越快，终局战争中越有优势

## 创建联盟

只有未加入联盟的玩家才能创建。创建后自动成为联盟创建者（最高权限）。

```json
{"type":"allianceAction","params":{
  "action":"CREATE_ALLIANCE",
  "allianceName":"联盟名称",
  "description":"联盟描述（可选）",
  "iconId":1
}}
```

- `allianceName` — 联盟名称，不可与现有联盟重复，不能包含违禁词
- `iconId` — 联盟图标编号（可选，默认1）

## 邀请玩家

只有联盟创建者才能邀请。被邀请的玩家会收到一封邮件，通过 `messageAction CHOICE_SELECT` 接受或拒绝。

```json
{"type":"allianceAction","params":{
  "action":"INVITE_PLAYER",
  "playerName":"目标玩家名"
}}
```

邀请策略：
- 优先邀请地理位置相近的活跃玩家（方便军事协作）
- 检查对方的排名和活跃度再决定邀请

## 离开联盟

非创建者可以离开联盟。离开会创建一个延时动作（不是立即生效）。

```json
{"type":"allianceAction","params":{
  "action":"LEAVE_ALLIANCE"
}}
```

- 创建者不能离开自己的联盟——需要先转让创建者身份或解散
- 正在离开中的玩家不能重复操作

## 外交协议

联盟创建者可以向其他联盟发起外交协议（同盟/中立/敌对等）。

```json
{"type":"allianceAction","params":{
  "action":"SEND_PACT_INVITATION",
  "allianceName":"目标联盟名称",
  "pactName":"协议类型标识"
}}
```

对方联盟创建者收到邮件后决定接受或拒绝。

## 联盟技能

联盟可以激活限时技能（每次只能运行一个，有冷却时间）。

```json
{"type":"allianceAction","params":{
  "action":"LAUNCH_ABILITY",
  "abilityTagName":"技能标识"
}}
```

- 只有联盟创建者才能激活
- 同时只能运行一个联盟技能
- 技能有冷却时间，激活后一段时间内无法再次使用

## 联盟影响力

- 联盟的总影响力 = 所有成员的个人影响力之和
- 个人影响力由城市数量、军事实力、领土面积等决定
- 影响力是终局战争（亚沙之泪）中争夺控制权的核心指标
- 联盟越大、成员越强，终局阶段越有优势

## 注意事项

- 联盟操作（创建/邀请/外交/技能）在世界进入限制状态时不可用
- 正在删除中的联盟不能执行任何操作
- 联盟成员上限由联盟等级决定
